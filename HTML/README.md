# Respaldo HTML de referencia

Esta carpeta conserva una copia de las plantillas HTML originales previas a la migración completa a PHP.

## Objetivo

- Servir como referencia visual/funcional histórica.
- Permitir comparar estructura y contenido contra las nuevas vistas renderizadas en PHP.

## Contenido

- `index.html`
- `foros.html`
- `contacto.html`
- `login.html`
- `dashboard-admin.html`
- `dashboard-asociado.html`
- `dashboard-usuario.html`
- `TEMPLATE OBJETIVO.html`

## Nota operativa

La aplicación en ejecución debe usar rutas PHP (`.php` o rutas resueltas por `public/index.php`) y **no depende** de estos archivos para funcionar en producción.
